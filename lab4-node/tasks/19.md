# Бот для салона красоты "BeautyBooker"
## Цель: Создать бота для записи и управления посещениями салона красоты.

### Функциональные требования:

1. Команда /book

Показывает доступные услуги и специалистов

Отображает свободное время для записи

Подтверждает бронирование и отправляет напоминание

2. Команда /services

Детализирует описание каждой услуги

Показывает цены и длительность процедур

Рекомендует дополнительные услуги

3. Команда /history

Показывает историю посещений

Сохраняет предпочтения клиента

Предлагает персональные акции и скидки
